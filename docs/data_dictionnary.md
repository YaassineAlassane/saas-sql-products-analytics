# Data Dictionary - SaaS Raw Layer

## Table: 
| Column | Type | Description |
| :--- | :--- | :--- |
|  | INTEGER | Unique identifier for each customer (Primary Key). |
|  | DATE | Original account creation date. |
|  | STRING | User's country of residence. |
|  | STRING | Subscription tier assigned at signup. |

## Table: 
| Column | Type | Description |
| :--- | :--- | :--- |
|  | INTEGER | Foreign key linking to the users table. |
|  | DATE | Date the subscription became active. |
|  | STRING | Billing country associated with the account. |
|  | STRING | Service level (Basic, Pro, Enterprise). |
|  | STRING | Current state of the account (Active, Cancelled). |
|  | INTEGER | Monthly revenue generated by the user. |
|  | DATE | Date the user cancelled the service (if applicable). |

## Table: 
| Column | Type | Description |
| :--- | :--- | :--- |
|  | INTEGER | ID of the user who performed the action. |
|  | TIMESTAMP | Precise date and time of the event. |
|  | STRING | Nature of the action (e.g., login, page_view). |
