# Data Dictionary - SaaS Financial Pipeline

## Table: raw_users (BigQuery Ingestion)
| Column | Type | Description |
| :--- | :--- | :--- |
| **user_id** | INTEGER | Unique identifier for each customer (Primary Key). |
| **created_at** | DATE | Original account creation date. |
| **country** | STRING | User's country of residence. |
| **signup_tier** | STRING | Subscription tier assigned at signup. |

## Table: raw_subscriptions (BigQuery Ingestion)
| Column | Type | Description |
| :--- | :--- | :--- |
| **user_id** | INTEGER | Foreign key linking to the users table. |
| **start_date** | DATE | Date the subscription became active. |
| **billing_country** | STRING | Billing country associated with the account. |
| **plan_type** | STRING | Service level (Basic, Pro). |
| **status** | STRING | Current state of the account (Active, Cancelled). |
| **mrr** | INTEGER | Monthly revenue generated by the user. |
| **end_date** | DATE | Date the user cancelled the service (if applicable). |

## Table: raw_events (Isolated / Future Extension)
| Column | Type | Description |
| :--- | :--- | :--- |
| **user_id** | INTEGER | ID of the user who performed the action. |
| **event_timestamp** | TIMESTAMP | Precise date and time of the event. |
| **event_type** | STRING | Nature of the action (e.g., login, page_view). |

## Table: fct_mrr (dbt Final Mart)
| Column Name | Data Type | Description |
| :--- | :--- | :--- |
| **user_id** | INTEGER | Unique identifier for the user. |
| **financial_month** | DATE | First day of the analyzed month (Grain: User × Month). |
| **mrr** | FLOAT | Validated Monthly Recurring Revenue. |
| **subscription_plan** | STRING | Active plan tier during the month (Basic / Pro). |
| **user_country** | STRING | Standardized country for geographic segmentation. |
