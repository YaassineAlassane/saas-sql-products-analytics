# SaaS Product Analytics - SQL Case Study

## 🎯 Business Context
This project focuses on analyzing customer behavior and financial performance for a B2B SaaS platform. As an **Analytics Engineer**, my role is to transform raw application data into actionable insights for stakeholders (Marketing, Sales, and Finance).

## 🚀 Objectives
- Master SQL for complex business logic (BigQuery).
- Calculate core SaaS metrics: **MRR, ARR, Churn Rate, and LTV**.
- Build a foundation for a future **dbt** (Data Build Tool) implementation.
- Prepare data for BI Dashboarding.

## 🛠 Tech Stack
- **Data Warehouse:** Google BigQuery
- **Language:** SQL (Standard SQL)
- **Version Control:** Git & GitHub

## 📂 Project Structure
- : SQL scripts organized by analytical depth.
- : Business logic definitions and data dictionary.
